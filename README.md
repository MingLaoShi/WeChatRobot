# WeChatRobot
## 简介
一个微信聊天机器人，使用pyhton制作，需要运行在windows电脑上。
## 运行环境
### 基本需求
python3.x
windows操作系统
微信客户端
### python库
pyautogui
PIL
uiautomation
openai
pyperclip
uuid
win32clipboard
## 实现功能
1.监控微信群聊天记录
将聊天记录记录在log文件中，成员可以发送带指令的消息来激活微信机器人。
2.ai聊天
链接openai的api来实现的ai实时聊天。
3.ai绘画
链接openai的api实现的ai图像生成。
4.排行榜
爬取网站的热门排行榜功能，发送到微信群中。
5.生成二维码
根据消息生成二维码。
6.表情合成
谷歌的表情合成。
